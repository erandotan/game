# 🎯 Deal Shot! — Presales World Tour Game

Multiplayer ball-throwing game for your happy hour!

## Deploy to Railway (FREE, 2 minutes)

1. **Create a GitHub repo**
   - Go to https://github.com/new
   - Create a new repo (e.g. `deal-shot`)
   - Upload all these files to it

2. **Deploy on Railway**
   - Go to https://railway.app
   - Sign in with GitHub (free)
   - Click **"New Project" → "Deploy from GitHub repo"**
   - Select your repo
   - Railway auto-detects Node.js and deploys!

3. **Get your public URL**
   - In Railway dashboard → your project → **Settings → Domains**
   - Click **"Generate Domain"**
   - You get a free URL like `https://deal-shot-production.up.railway.app`

4. **Share the URL on Zoom** — everyone just opens it in their browser!

## How to Play

- Everyone enters their name and joins
- **Host** clicks ▶ START when all players are ready
- Click & drag on the canvas to aim, release to throw
- Hit a deal hole → game pauses, spotlight moment! 🎤
- Host clicks ▶ RESUME to continue
- ↺ RESET to play again

## Run Locally

```bash
npm install
node server.js
# open http://localhost:3000
```
